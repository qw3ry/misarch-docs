# ShoppingCart Service

The shopping cart service provides the bounded context `ShoppingCart`.
It is responsible for managing shopping carts of users.

## Technology Stack

- **Language**: Rust
- **Framework**: [axum](https://docs.rs/axum/latest/axum/)
- **GraphQL Library**: [async_graphql](https://docs.rs/async-graphql/latest/async_graphql/)
- **Database**: MongoDB

## Defined Events

This section lists events that are defined or used by the shopping cart service.

### Subscribed Events

- [user/user/created](/docs/docs/services/user#userusercreated): Used to validate that shopping carts belong to an existing user
- [catalog/product-variant/created](/docs/docs/services/catalog#catalogproduct-variantcreated): Used to validate that product variants contained in shopping carts exist
- [order/order/created](/docs/docs/services/order#orderordercreated): Used to remove ordered items from the shopping cart
