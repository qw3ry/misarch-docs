# User Service

The user service provides the bounded context `User`.
It is responsible for managing users by interacting with Keycloak.

## Domain Model

<CustomImage path="/renderedDiagrams/userDomainModel" width="685.5" height="381.5" />

## Technology Stack

- **Language**: Kotlin
- **Framework**: Spring Boot
- **GraphQL Library**: GraphQL Kotlin
- **Database**: PostgreSQL
- **Schema Management**: Flyway

## Events

This section lists events that are defined or used by the user service.

### Published Events
<details>
<summary>
#### user/user/created
</summary>

This event is published when a new user is created.

```tsx
{
  id: string,
  username: string,
  name: Name,
  dateJoined: string
}
```
</details>

### Subscribed Events

<details>
    <summary>
#### user/user/create
    </summary>

Published by Keycloak to inform the user service about a newly created user account.

```tsx
{
  id: string,
  username: string,
  firstName: string,
  lastName: string
}
```
</details>
