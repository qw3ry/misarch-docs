# Review Service

The review service provides the bounded context `Review`.
It is responsible for managing reviews of users.

## Technology Stack

- **Language**: Rust
- **Framework**: [axum](https://docs.rs/axum/latest/axum/)
- **GraphQL Library**: [async_graphql](https://docs.rs/async-graphql/latest/async_graphql/)
- **Database**: MongoDB

## Events

This section lists events that are defined or used by the review service.

### Subscribed Events

- [user/user/created](/docs/docs/architecture/services/user#userusercreated): Used to validate that reviews belong to an existing user
- [catalog/product-variant/created](/docs/docs/architecture/services/catalog#catalogproduct-variantcreated): Used to validate that product variants contained in reviews exist
