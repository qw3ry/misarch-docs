# Notification Service

The notification service provides the bounded context `Notification`.
It is responsible for managing notifications.

## Technology Stack

- **Language**: Kotlin
- **Framework**: Spring Boot
- **GraphQL Library**: GraphQL Kotlin
- **Database**: PostgreSQL
- **Schema Management**: Flyway

## Events

This section lists events that are defined or used by the notification service.

### Subscribed Events

- **notification/notification/create**: Other services can publish this event to create new notifications.
    ```tsx
    {
      title: string,
      body: string,
      userId: boolean
    }
    ```

- [user/user/created](/docs/docs/services/user#userusercreated): Used to validate that only existing users can receive notifications
