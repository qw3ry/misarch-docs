# Invoice Service

The invoice service provides the bounded context `Invoice`.
It is responsible for managing invoices of orders.

## Events

This section lists events that are defined or used by the invoice service.

### Published Events

<details>
    <summary>
#### invoice/invoice/created
    </summary>

This event is published when an invoice for an order is created.

```tsx
{
  order: Order,
  invoice: Invoice,
}
```
</details>

### Subscribed Events

- [discount/order/validation-succeeded](/docs/docs/services/discount#discountordervalidation-succeeded): Triggers this service to create an invoice and emit following event: `invoice/invoice/created`
- [address/vendor-address/created](/docs/docs/services/address#addressvendor-addresscreated): Used to obtain vendor address.
- [user/user/created](/docs/docs/services/user#userusercreated): Used to synchronize user addresses on users.
- [address/user-address/created](/docs/docs/services/address#addressuser-addresscreated): Used to obtain user address.
- [address/user-address/archived](/docs/docs/services/address#addressuser-addresscreated): Used to obtain user address.
